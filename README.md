# Prueba 1 — Programación Web

Breve descripción
------------------
Primera prueba para la asignatura de Programación Web. Sitio estático sencillo creado usando HTML y CSS puro.

Tecnologías utilizadas
----------------------
- HTML5 (archivo index.html)
- CSS3 puro (archivo style.css)

Estructura del proyecto
-----------------------
- index.html — página principal.
- style.css — estilos de la página.
- recursos/ — imágenes, fuentes u otros activos usados por la web.

Cómo abrir
----------
1. Abrir el archivo [index.html](JordanMurillo-AdrianRamirez_Eva_1/index.html) en un navegador (doble clic o arrastrar al navegador).
2. No requiere servidor ni dependencias externas.

Notas
-----
- Trabajo realizado como prueba práctica de la asignatura; objetivo: practicar la maquetación con HTML y estilos con CSS sin frameworks.
